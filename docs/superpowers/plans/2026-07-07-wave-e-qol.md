# Wave E — QoL Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Three quality-of-life features: dictionary word starring (priority bias), settings/dictionary/snippets export-import, and a floating scratchpad window that captures dictation without disturbing the focused app.

**Architecture:** Starring is a `*` prefix convention handled entirely in the pure `dictionary.ts` parser (starred words sort first so they survive the 600-char bias cap). Export/import adds two IPC handlers using Electron `dialog` + a pure `sanitizeImport` validator in `config.ts`. The scratchpad is a fourth BrowserWindow (frameless, always-on-top, design-B charcoal) whose main-process module owns content persistence and a "capture" flag; the pipeline gains an optional `routeText` dep checked just before `inject()` — when the scratchpad is capturing, dictation text routes there instead of pasting into the focused app.

**Tech Stack:** Electron 39 (TypeScript strict), electron-vite, vitest, electron-store v8. No new dependencies.

## Global Constraints

- Never-throw/never-block contracts: scratchpad routing and export/import must never break the dictation hot path; any failure falls back to normal inject.
- All renderer styling matches design B: charcoal `#1b1b1d` surfaces, red reserved for single accents (active states, primary buttons), JetBrains Mono/Space Grotesk fonts as used in `settings.html`.
- `settings.dictionary` and `settings.snippets` remain `string[]` — no schema migration; starring is an in-string convention (`*` prefix).
- Export JSON includes API keys (personal-backup tool, private machine) — the import validator must accept files with or without any subset of keys.
- New IPC channels follow the `IPC` const pattern in `app/src/shared/types.ts` (camelCase key → `'namespace:action'`).
- Every task: tests green (`npx vitest run`), then commit. Run from `app/`.
- Repo root: `C:\Users\owen\Downloads\OwenFlow`. App source in `app/src`. NEVER touch `app/out` or `app/dist`.

A dictionary entry starting with `*` is "starred": the `*` is stripped, and starred prompt words are ordered BEFORE unstarred ones in `promptWords` (so they always survive the 600-char cap).
```ts
ipcMain.handle(IPC.settingsExport, async () => {
  try {
    const { canceled, filePath } = await dialog.showSaveDialog({
      title: 'Export OwenFlow settings',
      defaultPath: `owenflow-settings-${new Date().toISOString().slice(0, 10)}.json`,
      filters: [{ name: 'JSON', extensions: ['json'] }]
    })
    if (canceled || !filePath) return { ok: false, error: 'canceled' }
    const payload = {
      app: 'owenflow',
      version: app.getVersion(),
      exportedAt: new Date().toISOString(),
      settings: getSettings()
    }
    await writeFile(filePath, JSON.stringify(payload, null, 2), 'utf8')
    return { ok: true, path: filePath }
  } catch (err) {
    return { ok: false, error: err instanceof Error ? err.message : String(err) }
  }
})

ipcMain.handle(IPC.settingsImport, async () => {
  try {
    const { canceled, filePaths } = await dialog.showOpenDialog({
      title: 'Import OwenFlow settings',
      filters: [{ name: 'JSON', extensions: ['json'] }],
      properties: ['openFile']
    })
    if (canceled || !filePaths[0]) return { ok: false, error: 'canceled' }
    const parsed = JSON.parse(await readFile(filePaths[0], 'utf8')) as unknown
    const patch = sanitizeImport(parsed)
    const keys = Object.keys(patch)
    if (keys.length === 0) return { ok: false, error: 'no valid settings found in file' }
    setSettings(patch)
    return { ok: true, applied: keys.length }
  } catch (err) {
    return { ok: false, error: err instanceof Error ? err.message : String(err) }
  }
})
```

Use `node:fs/promises` `readFile`/`writeFile`. Note: `setSettings` already triggers the existing settings-changed side effects (hotkey re-register, sidecar restart on model change) — no extra work.
